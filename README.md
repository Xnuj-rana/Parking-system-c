# 🚗 Smart Parking Lot System in C

A simple C language project that simulates a real-world parking lot system using arrays and structures.
The program allows users to park vehicles, remove them, search for vehicles, and check slot availability efficiently.

---

## 📜 Features
- Automatic parking slot allocation
- Vehicle removal system
- Search vehicle by number
- Display parking status
- Show available slots
- Simple and beginner-friendly code

---

## ⚙️ How to Run
1. Copy or download the code file: parking_lot.c
2. Open it in any C compiler (VS Code, OnlineGDB, Turbo C, etc.)
3. Compile and run it.
